# LingoStream

LingoStream is a mobile app for learning languages while watching movies, clips and songs.

## Main idea

A karaoke-style subtitle and translation window helps users learn while watching content.

## Current MVP

- Import `.srt` subtitles
- Karaoke-style text display
- Movable translation bubble
- Local dictionary
- Learning direction:
  - English → Ukrainian
  - Ukrainian → English
- Basic catalog of YouTube learning content
- Free/PRO structure prepared
- GitHub Actions APK build

## Monetization plan

### Free
- SRT import
- Karaoke window
- Basic dictionary
- Ads placeholder

### PRO
- No ads
- Unlimited dictionary
- Advanced repeat mode
- Extended catalog
- Future overlay mode
